# yth-event-notifier
telegram bot to notify users of upcoming events in the yth calander (excel)

## installation
```bash
git clone https://github.com/Donut100/yth-event-notifier.git
cd yth-event-notifier
```
```py
pip install -r requirement.txt

# for windows
python main.py

# for MacOs / Linux
python3 main.py
```
